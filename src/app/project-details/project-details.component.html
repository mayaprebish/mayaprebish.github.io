In this group project for CS4550 (Web Development), my partners and I created a full-stack web application using React/js, Redux/js, Spring Boot, REST, Java, Bootstrap, and the Eventful API. The application is useful for two types of users:
  1. Tour Managers: A tour manager can add artists they manage to their profile, create tours, and then search for venues to add to their tours for artists from their profile.
  2. Venue Owners: A venue owner can add venues to their profile, and can see when a tour manager has 'booked' their venue for a tour.
My main role in this project was wiring up data from back-end to front-end: I wrote Redux code to maintain state of data on the front-end, and wrote REST API calls to fetch data from our back-end. I also helped to design and implement back-end code in Java.

Partners: Morolayo Lambo, Amanda Escaba, Jada Letts

For this program, I learned the basics of MATLAB in order to produce a Live Script that researchers can use to track movement over specific periods of time using a table of data.
